## 2-Minute Lab Kickoff Script

1. Show blank template (30–40s).
2. Show expected output quality criteria (40–50s).
3. State 3 quality criteria teams must match.
4. Start timer and announce midpoint + 2-minute warning.

## Escalation / Fallbacks

- No admin access: switch to policy simulation path with screenshots.
- Secret scan delay: use sample alert evidence and proceed with remediation workflow.
- Copilot issues: pair participant, continue using backup account.
- Overrun: shorten lecture, do not skip validation gate.

## End-of-Day Exit Criteria

- All teams submitted all mandatory artifacts.
- Each artifact includes owner, date, and next action.
- Teams have a 30-day follow-up checkpoint date.
- Open blockers captured with responsible owner.
